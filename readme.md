> [!NOTE]
> В папках `typst` я дополнительно реализовал предложенные задачи на Typst, потому что я твердо убежден: мы должны перестать использовать LaTeX и переходить на Typst прямо сейчас!

LaTeX файлы лежат в соответствующих папках `latex` каждого задания.

> [!NOTE]
> Поскольку в Typst нет жестко заданных классов документов (`article`, `book` и т.д.), в Задании 1 представлен только один файл Typst со стилями по умолчанию.

Компиляция LaTeX файлов для заданий `task1` и `task2` происходит командой `latexmk <path to main LaTeX file>`
Компиляция Typst файлов для заданий `task1`, `task2` и `task3` происходит командой `typst compile <path to main Typst file>`

Компиляция LaTeX из `task3` осуществляется переходом в папку `task3/latex` и использованием команды `make all`

- `task1` — задание на форматирование текста 
- `task2` — задание на набор математических формул
- `task3` — задание на использование шаблона

Шаблон для курсовой работы на LaTeX взят из репозитория https://github.com/csit-sgu/latex-template

Шаблон для курсовой работы на Typst взят из репозитория https://github.com/danilasar/typst-template
